Mapa Współwystępowania Gatunków
Linie łączą gatunki, które często pojawiają się razem — bo spełniają się dla nich te same warunki środowiskowe. Grubość linii = siła powiązania.

❄️ Niska temperatura + Stary osad

Microthrix parvicella Type 0092 Nocardia spp.
M. parvicella dominuje zimą. Type 0092 preferuje ciepło, ale przy długim SRT pojawiają się razem.

🧈 Wysokie stężenie FOG (tłuszcze)

Nocardia spp. Microthrix parvicella Type 0092
Wszystkie trzy gatunki mają hydrofobowe ściany komórkowe ułatwiające asymilację lipidów.

💀 Septyczność + Siarczki

Thiothrix spp. Type 021N Beggiatoa*
Zagniłe ścieki z siarczkami = idealne warunki dla bakterii siarkowych. Klasyczne trio.

📉 Niskie obciążenie F/M

Type 0041 Type 0675 Type 1851 Microthrix parvicella
Klasyczna „czwórka biedy" przy niedociążeniu: wolna kinetyka wzrostu, efektywne wyłapywanie substratów śladowych.

⬇️ Niedotlenienie (DO < 1 mg/L)

Thiothrix spp. Haliscomenobacter hydrossis Sphaerotilus natans Type 1701
H. hydrossis to precyzyjny detektor deficytu tlenu. Thiothrix i Sphaerotilus dołączają przy pogłębionym niedotlenieniu.

🧪 Niedobór biogenów (N i P)

Type 021N Type 0041 Nostocoida limicola
Type 021N to silny wskaźnik niedoboru azotu. Nostocoida reaguje na deficyt N, P i obecność kwasów lotnych.
